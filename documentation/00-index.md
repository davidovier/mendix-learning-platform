## Top 10 Most Tested Topics

Based on 268 actual exam questions:

1. **Microflow prefixes and naming** (DS_, ACT_, SUB_, VAL_, etc.)
2. **List operations** (head, tail, find, filter, union, intersect, subtract)
3. **Entity types and colors** (blue=persistable, orange=NPE, etc.)
4. **Association ownership** (NPE must own association to persistable)
5. **Delete behavior** (red=cascade, blue=conditional)
6. **Security access locations** (App Security vs Module Security)
7. **XPath syntax** (brackets, tokens, operators)
8. **Scrum values and events**
9. **Reference selectors** (which widget for which relationship)
10. **Anonymous user configuration**

## Key Visual Indicators to Memorize

### Entity Colors
| Color | Type |
|-------|------|
| Blue | Persistable |
| Orange | Non-Persistable |
| Purple | External |
| Green | View Entity |

### Delete Behavior
| Border | Behavior |
|--------|----------|
| Red | Cascade delete |
| Blue | Conditional delete |

### Data Hub Lines
| Line | Meaning |
|------|---------|
| Dotted | Consuming |
| Solid | Providing |

## Exam Information

- **Questions**: 50
- **Time**: 90 minutes
- **Format**: Online proctored, multiple choice
- **Cost**: €148 / £124 / $158
- **Prerequisite**: Rapid Developer certification
- **Pass requirement**: ~70% (approximately 35 correct answers)
