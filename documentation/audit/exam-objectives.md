# Mendix Intermediate Developer Exam Objectives

## Exam Format
- **Questions:** 50 multiple choice
- **Time:** 90 minutes
- **Format:** Online proctored, on-demand
- **Cost:** €148 / £124 / $158
- **Prerequisite:** Rapid Developer certification
- **Pass Score:** ~70% (approximately 35 correct)

## Certification Level Description
The Intermediate Developer certification proves that your "knowledge and skills extend beyond the fundamentals of the Mendix platform. You are ready to build more complex features but still need the occasional guidance from more advanced developers."

## Core Topics (from Studio Pro Reference Guide)

Based on official Mendix documentation structure, the intermediate exam covers:

### 1. Domain Model & Data Management
- Entities, attributes, and associations
- Validation rules
- Generalization and specialization
- Persistable vs Non-Persistable entities
- System members (createdDate, changedDate, owner, changedBy)
- Delete behavior (cascade, prevent)

### 2. Microflows
- Activities (object, list, variable, client)
- Decisions and merges
- Loops (iterate, while)
- Error handling (rollback, custom handling)
- Parameters and return values
- Debugging with breakpoints

### 3. Nanoflows
- Client-side execution
- Offline-first patterns
- Differences from microflows

### 4. Pages & UI
- Page properties and parameters
- Data containers (data views, list views, grids)
- Input widgets
- Layouts and snippets
- Reference selectors

### 5. Security
- App-level vs Module-level security
- User roles and module roles
- Entity access rules
- XPath constraints in security
- Anonymous users

### 6. XPath & Expressions
- XPath syntax and operators
- Tokens ([%CurrentUser%], [%CurrentDateTime%])
- Expression functions
- Constraints in retrieves and access rules

### 7. Integration
- REST services (consumed and published)
- OData services
- Import/Export mappings
- JSON and XML handling

### 8. Modules
- Module structure and organization
- App Explorer folders
- Cross-module associations

### 9. Java & JavaScript Extensions
- Java actions
- Custom widgets
- Extending platform functionality

### 10. Events & Validation
- Event handlers (before/after commit, create, delete)
- Validation rules in domain model

### 11. Enumerations & Constants
- Enumeration values and captions
- Constants for configuration

### 12. Scheduled Events
- Automated microflow execution
- Scheduling configuration

### 13. Version Control
- Git operations
- Branching and merging
- Conflict resolution
- Tagged versions

### 14. Agile/Scrum
- Sprint events (planning, daily, review, retrospective)
- User stories and story points
- Scrum roles and values
